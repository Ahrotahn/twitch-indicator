# Twitch Indicator

![](http://i.imgur.com/zWfcJuJ.png)

Twitch.tv indicator for Linux. Tracks your followed channels and notifies when they go live.

### Setup

```
cd twitch-indicator
sudo ./setup.sh
twitch-indicator &
```
